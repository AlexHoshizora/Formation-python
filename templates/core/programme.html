Programme détaillé

Le programme complet, module par module.

Voici ce que vous allez découvrir au fil des 11 étapes de la formation : chaque module a un objectif clair, des explications structurées et de la pratique pour ancrer les notions.

Module 0

Découvrir Python et installer tout ce qu’il faut

On prépare votre environnement pour que vous puissiez lancer du code Python sans prise de tête.

  • Télécharger et installer Python 3 proprement sur votre machine
  • Installer un éditeur de code comme VS Code et activer la coloration Python
  • Lancer Python en ligne de commande (python --version, mode interactif) et via un fichier .py
  • Écrire et exécuter votre tout premier programme print("Bonjour")
Module 1

Manipuler des nombres et des textes sans se tromper

Vous apprenez ce qu’est une “variable” et comment stocker des informations simples en Python.

  • Créer des variables de types de base : int, float, str, bool
  • Utiliser les fonctions intégrées input() et print() pour dialoguer avec l’utilisateur
  • Convertir proprement un texte saisi en nombre avec int() ou float() (pour faire des calculs)
  • Assembler des textes et des valeurs avec des f‑strings pour afficher des messages lisibles
Module 2

Apprendre à faire des choix dans votre code

Votre programme réagit différemment selon les situations grâce aux conditions.

  • Écrire des blocs conditionnels avec if, elif, else
  • Comparer des valeurs avec les opérateurs >, <, ==, != et combiner avec and / or
  • Écrire un script qui vérifie l’âge de l’utilisateur (majeur / mineur) ou contrôle un mot de passe simple
  • Créer un petit menu texte à options (1, 2, 3…) qui appelle différentes parties du code
Module 3

Répéter des actions et manipuler des listes d’éléments

Vous ne recopiez plus dix fois la même ligne : vous laissez la machine faire la répétition.

  • Utiliser des boucles for et while sans tomber dans les boucles infinies
  • Créer et modifier des listes (list) pour stocker plusieurs valeurs
  • Découvrir les dictionnaires (dict) pour associer une “clé” à une “valeur” (ex : nom → numéro de téléphone)
  • Écrire un script qui parcourt une liste et calcule un total, une moyenne ou filtre certains éléments
Module 4

Découper votre code en petits blocs réutilisables

Vous transformez des morceaux de code en “outils” réutilisables grâce aux fonctions.

  • Définir vos propres fonctions avec def, des paramètres et une valeur renvoyée avec return
  • Comprendre la différence entre une fonction qui affiche un résultat et une fonction qui retourne une valeur
  • Écrire une fonction qui calcule un prix avec réduction, ou une moyenne de notes à partir d’une liste
  • Commencer à séparer la logique en plusieurs fonctions pour clarifier le flux de votre programme
Module 5

Comprendre ce qui se passe “dans les coulisses”

On explique enfin pourquoi certaines modifications semblent se propager “toutes seules” dans votre code.

  • Voir la différence entre types immutables (nombres, textes) et types mutables (listes, dictionnaires)
  • Comprendre l’idée de référence en mémoire et pourquoi deux variables peuvent pointer vers la même liste
  • Apprendre à copier une liste correctement avec list(), copy() ou la notation [:]
  • Découvrir quelques bugs classiques liés à la mutabilité et les corriger étape par étape
Module 6

Ne plus paniquer devant les messages d’erreur

Les erreurs deviennent des informations utiles plutôt que des murs incompréhensibles.

  • Lire un traceback Python et repérer la ligne vraiment problématique
  • Utiliser try / except pour intercepter des exceptions courantes (ValueError, TypeError…)
  • Protéger les saisies utilisateur pour éviter les plantages (saisie vide, texte au lieu de nombre, etc.)
  • Afficher des messages d’erreur clairs et compréhensibles pour l’utilisateur final
Module 7

Lire et écrire dans des fichiers (CSV, textes…)

Votre code commence à automatiser de petites tâches répétitives autour de fichiers.

  • Ouvrir un fichier avec open() et le manipuler en utilisant le mot‑clé with
  • Parcourir un fichier ligne par ligne pour analyser ou filtrer son contenu
  • Utiliser le module standard csv pour lire un tableau de données (CSV simple)
  • Générer automatiquement un rapport texte ou un fichier CSV à partir de données traitées en Python
Module 8

Organiser vos données comme des “objets” du quotidien

Vous découvrez doucement la programmation orientée objet, avec des exemples très concrets.

  • Créer une classe simple avec class pour représenter un “client”, un “livre” ou une “commande”
  • Ajouter des attributs (données) et des méthodes (actions) à vos objets
  • Mettre en place un constructeur __init__() pour initialiser correctement un objet
  • Voir dans quels cas la POO (programmation orientée objet) rend votre code plus clair et plus évolutif
Module 9

Écrire du code lisible, même dans 6 mois

Vous prenez des habitudes de “vrai développeur” pour que votre code reste clair dans le temps.

  • Découper un projet en plusieurs fichiers / modules Python et utiliser import
  • Adopter quelques règles simples de la convention PEP 8 (noms, indentation, longueur de ligne…)
  • Nommer clairement vos variables, fonctions et classes pour qu’un autre développeur comprenne votre intention
  • Nettoyer un script existant (avant / après) pour voir concrètement l’impact sur la lisibilité
Module 10

Mettre tout ensemble dans de vrais petits projets

Vous appliquez toutes les notions vues dans la formation à des projets complets, utiles et motivants.

  • Créer un petit jeu ou quiz en ligne de commande (par exemple : jeu de devinette de nombre)
  • Construire un mini gestionnaire (tâches, livres, contacts…) en combinant listes, dictionnaires, fonctions et fichiers
  • Écrire un script d’automatisation qui parcourt un dossier, lit plusieurs fichiers et génère un récapitulatif
  • Revoir le parcours complet et établir votre feuille de route (projets perso, bibliothèques, frameworks web…)
