Module 0 · Découvrir Python et installer tout ce qu’il faut

Installer Python 3 correctement sur votre ordinateur

Avant d’écrire une seule ligne de code, il est essentiel que votre environnement soit propre et fiable. Dans cette leçon, vous allez :

  • vérifier si Python est déjà présent sur votre machine,
  • installer une version récente de Python 3 si nécessaire,
  • vous assurer que la commande python ou python3 fonctionne où que vous soyez.

L’idée n’est pas seulement de “cliquer sur suivant”, mais de comprendre ce que vous faites pour que, le jour où quelque chose ne fonctionne plus, vous sachiez où regarder et quoi vérifier.

1. Vérifier si Python est déjà installé

De nombreuses machines ont déjà une version de Python installée, surtout sous macOS et Linux. Avant d’ajouter une nouvelle installation, on commence donc par faire un état des lieux.

  • Ouvrez un Terminal (macOS / Linux) ou une Invite de commandes (Windows).
  • Tapez : python --version puis appuyez sur Entrée.
  • Si vous obtenez un message d’erreur ou une version en 2.x, essayez : python3 --version.

Si vous voyez une version commençant par 3. (par exemple Python 3.12.0), cela signifie que Python 3 est déjà installé et accessible. Nous allons quand même le relancer un peu plus loin pour vérifier que tout fonctionne bien.

Si la commande n’est pas reconnue : ce n’est pas grave. Cela signifie simplement que Python n’est pas encore installé, ou pas accessible depuis la ligne de commande. Vous allez corriger cela dans la partie suivante.

2. Installer Python 3 sur Windows

Sur Windows, la méthode la plus simple et la plus fiable consiste à passer par le site officiel de Python. Vous allez télécharger un installeur, le lancer, puis cocher une option importante pour que Python soit accessible partout.

  1. Ouvrez votre navigateur et allez sur python.org/downloads.
  2. Cliquez sur le gros bouton jaune qui propose la dernière version de Python 3 pour Windows.
  3. Une fois le fichier téléchargé, double‑cliquez dessus pour lancer l’installation.
  4. Sur l’écran d’accueil de l’installeur, cochez la case “Add Python to PATH”. C’est ce qui permet de lancer Python depuis n’importe quel dossier dans l’invite de commandes.
  5. Cliquez sur Install Now et laissez l’installation se terminer.

Une fois l’installation terminée, fermez l’installeur, rouvrez une nouvelle invite de commandes et testez : python --version. Vous devriez maintenant voir apparaître une version 3.x.

3. Installer Python 3 sur macOS

macOS fournit parfois une ancienne version de Python par défaut. Pour travailler sereinement, vous allez installer une version récente de Python 3 en parallèle, via un installeur officiel.

  1. Ouvrez votre navigateur et rendez‑vous sur python.org/downloads/macos.
  2. Téléchargez le fichier d’installation correspondant à la dernière version de Python 3.
  3. Ouvrez le fichier .pkg téléchargé, puis suivez les étapes de l’installateur (continuer, accepter, installer…).
  4. À la fin de l’installation, ouvrez l’application Terminal depuis votre dossier Applications.
  5. Tapez python3 --version et vérifiez que vous voyez une version 3.x.

Sur macOS, il est courant d’utiliser la commande python3 au lieu de python. Dans la suite de la formation, quand vous verrez python, retenez que sur macOS vous pouvez souvent remplacer par python3.

4. Vérifier concrètement que tout fonctionne

Maintenant que Python est installé, l’objectif est de vérifier que vous pouvez lancer une session interactive et exécuter un petit morceau de code. C’est votre “test de santé” Python.

  • Ouvrez votre terminal (macOS / Linux) ou votre invite de commandes (Windows).
  • Tapez python (ou python3 sur macOS si nécessaire) puis appuyez sur Entrée. Vous devriez voir apparaître quelques lignes d’information, puis un prompt qui ressemble à >>.
  • Tapez ensuite : print("Bonjour") puis validez. Si le mot Bonjour s’affiche sur la ligne suivante, c’est que Python exécute bien votre code.
  • Pour quitter cette session interactive, tapez exit() puis Entrée (ou utilisez le raccourci Ctrl + Z sous Windows, Ctrl + D sous macOS / Linux).

Vous venez d’écrire et d’exécuter votre première instruction Python. Ce n’est pas encore un “programme” au sens d’un fichier, mais c’est une étape importante : vous savez maintenant que Python répond bien à ce que vous lui demandez.

5. Problèmes fréquents et pistes de résolution

Si tout ne fonctionne pas du premier coup, ce n’est pas un échec, c’est l’occasion de comprendre un peu mieux ce qui se passe. Voici les cas les plus courants et comment les traiter.

  • “python n’est pas reconnu” sous Windows : cela signifie que Windows ne sait pas où se trouve Python. Réinstallez Python en vous assurant de cocher la case Add Python to PATH, puis rouvrez une nouvelle invite de commandes et réessayez.
  • Plusieurs versions de Python installées : si python lance une version 2.x et python3 une version 3.x, utilisez systématiquement python3 pour cette formation afin d’éviter les confusions.
  • Rien ne s’affiche après avoir tapé une commande : vérifiez bien que vous avez appuyé sur Entrée, et que vous êtes dans un terminal / une invite de commandes, pas dans une autre application.
Si vous êtes parvenu à ouvrir Python, à exécuter print("Bonjour") puis à quitter proprement la session, cette étape est validée. Vous pouvez passer sereinement au point suivant : installer et configurer votre éditeur de code pour travailler dans de bonnes conditions.
